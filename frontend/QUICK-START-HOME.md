## 🐛 故障排查

### 如果页面不显示

#### 1. 检查服务器状态
```powershell
Get-Process | Where-Object { $_.ProcessName -like "*node*" }
```
应该看到 Node.js 进程正在运行

#### 2. 重启开发服务器
```powershell
# 在 frontend 目录
cd d:\Projects\2025\auto-score-analyzer\frontend
npm run dev
```

#### 3. 清除浏览器缓存
- 按 `Ctrl + Shift + R` 强制刷新
- 或清除浏览器缓存后重新访问

### 如果样式不正常

#### 检查 CSS 文件是否存在
```powershell
Test-Path "d:\Projects\2025\auto-score-analyzer\frontend\src\styles\home.css"
# 应该返回 True
```

#### 检查 import 语句
Home.tsx 第 7 行应该有：
```typescript
import '../styles/home.css';
```
