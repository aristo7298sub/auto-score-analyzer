# 你的最终目的，是制作一个学生成绩分析网站，具体需求如下：

## 1. 该网站支持上传excel / word / ppt 等主流办公文档，并提取文件中的学生成绩信息；
### 1.1 示例文件如该文件夹中的"""副本分析表1.xlsx"""，第一行为知识点，第一列从第二行开始为学生姓名，学生姓名所在的行，表示该学生的扣分项，不代表真实扣分，如没有值，则为不扣分。最后一列为总分。其他配置完全保留。

## 2. 文件上传后，后端需要提取并解析文件中的学生成绩信息，然后发送给Azure OpenAI接口，Azure OpenAI接口将会总结某个学生失分在哪里，以及针对这些扣分项有哪些提升建议。
### 2.1 我的理解是，文件上传后，需要为每个学生的扣分项创建数组，但如果你有其他方式汇总这些数据，也没有问题。

## 3. 改网站支持展示每个学生扣分项的总结，以及针对这些扣分项有哪些提升建议。

## 4. 该网站也支持学生姓名搜索，输入学生姓名后，网站会返回该学生的扣分项，以及针对这些扣分项有哪些提升建议。

## 5. 改网站支持可视化，比如柱状图，折线图，饼图等，展示每个学生扣分项的总结，以及针对这些扣分项有哪些提升建议。

## 6. 该网站支持导出，导出为excel / word / pptx 等主流办公文档。导出文件中，需要包含每个学生的姓名，扣分项的总结，以及针对这些扣分项有哪些提升建议。

# 项目请选择使用javascript或者python来实现，尽可能少的依赖第三方库，在实现需求的基础上，让项目尽可能简单。另外，因为项目涉及多个模块，建议模块化编程，方便未来升级需求。
